# Gift Image Overrides / Замены картинок подарков

EN: Place image files named `<slug>.{png,webp,jpg}` here to override
    the default gift image downloaded from API.
    These files take priority over `gift_cache/` downloads.

RU: Положите сюда файл `<slug>.{png,webp,jpg}`, чтобы заменить
    картинку подарка, скачанную с API.
    Эти файлы имеют приоритет над `gift_cache/`.

Example / Пример: `ufcstrike.webp`
